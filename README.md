# C/C++ Programs

```shell
g++ -o prime23 prime23.cpp
```

Run with: 
```shell
./prime23
```

## Lab03.cpp

Program to determine the value of the voltage gain after user inputs the number of stages and the frequency in an amplifier.

If in root directory, navigate to pt/Lab03/ with:
```shell
cd pt/Lab03/
```

Compile with: 
```shell
make Lab03
```

Run with: 
```shell
./Lab03
```
